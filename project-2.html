This project was developed during a technical course in Internet Informatics to apply the use of APIs and enhance JavaScript skills. It is a game that plays songs from Disney animations, and the player must guess which movie the song is from. The project was showcased several times as a promotional tool for Etec Darcy Pereira de Moraes.
Designed by Ana Carolina Souza Barros.

preview

The initial project developed in class used Spotify's Web Playback SDK API, enabling music playback on a website. Access version 1.0 featuring API integration.
Version 2.0 introduces an option with locally hosted songs, allowing for easy playlist customization and offline gameplay. Access this version online.
